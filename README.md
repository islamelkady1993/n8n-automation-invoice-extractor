# n8n-automation-invoice-extractor

## 🚀 AI Invoice Extractor & Automation Workflow (n8n + OpenAI)

مشروع أتمتة كامل لسحب البيانات من الفواتير (PDF/Images) باستخدام الذكاء الاصطناعي وتنظيمها تلقائياً.

### 📋 نظرة عامة (Overview)
هذا المستودع يحتوي على سير عمل (Workflow) تم بناؤه باستخدام أداة **n8n**. يقوم النظام بمراقبة ملفات Google Drive، وعند رفع أي فاتورة، يقوم الذكاء الاصطناعي (OpenAI) بتحليل النص واستخراج البيانات الأساسية وتحويلها إلى تنسيق JSON المنظم، ثم يتم تخزينها في Google Sheets وإرسال إشعار عبر البريد الإلكتروني.

### 🛠️ التقنيات المستخدمة (Tech Stack)
- **n8n**: كأداة أساسية لربط الخدمات (Automation).
- **OpenAI (GPT-4o)**: لتحليل النصوص باللغة العربية والإنجليزية بدقة عالية.
- **Google Drive**: كمصدر لاستقبال الفواتير.
- **Google Sheets**: لقاعدة بيانات الفواتير.
- **Gmail**: لإرسال التقارير التلقائية.<img width="1023" height="632" alt="Screenshot 2026-03-18 040024" src="https://github.com/user-attachments/assets/9a731100-e795-47ca-a04f-023bb4931b68" />


### ⚙️ المميزات (Features)
- **دعم اللغتين**: استخراج البيانات من الفواتير العربية والإنجليزية بكفاءة.
- **دقة البيانات**: سحب رقم الفاتورة، التاريخ، الإجمالي، واسم الموظف/المحل بدقة.
- **توفير الوقت**: تحويل العملية اليدوية لمدخلات البيانات إلى عملية آلية 100%.

### 🚀 كيفية الاستخدام (How to Use)
1. قم بتحميل ملف الـ `.json` الموجود في هذا المستودع.
2. افتح واجهة n8n الخاصة بك واضغط على **Import from File**.
3. قم بتوصيل حساباتك (Credentials) لكل من: OpenAI, Google Drive, Google Sheets, Gmail.
4. اضغط على **Execute Workflow** وابدأ في رفع الفواتير!
